## 📖 Descripción
El presente proyecto ha sido desarrollado como parte de mi proceso de aprendizaje en Python, donde he podido aplicar  onceptos básicos de este lenguaje en una aplicación interactiva creada con Streamlit simulando acciones que puedo presenciar en mis actividades laborales. 

🔎 En este proyecto trabajé con:

- ✅ Variables y tipos de datos
- ✅ Estructuras de datos (listas y diccionarios)
- ✅ Funciones y condicionales
- ✅ Manejo de estado con st.session_state
- ✅ Desarrollo de una interfaz interactiva con Streamlit

A través de este proyecto he podido consolidar mis bases en programación y comprendí cómo se logra estructurar una aplicación sencilla en Python, enfocándome en la claridad del código y la organización del proyecto.

## 🎯 Objetivos
- Aplicar los fundamentos de Python en un proyecto práctico, reforzando el uso de estructuras de datos, funciones y lógica condicional.
- Desarrollar una aplicación interactiva con Streamlit, comprendiendo cómo conectar la lógica del programa con una interfaz visual.
- Consolidar buenas prácticas de organización y estructura de código, sentando una base sólida para proyectos más avanzados en el futuro.
